## 🚀 Quick Installation

### Termux Installation
```bash
wget -O wp-self-panel.sh https://raw.githubusercontent.com/likhonsheikhcodes/WordPressSelfPanel/main/install.sh && bash wp-self-panel.sh
```

### Vercel Deployment
```bash
# Clone Repository
git clone https://github.com/likhonsheikhcodes/WordPressSelfPanel.git

# Deploy
vercel --prod
```
