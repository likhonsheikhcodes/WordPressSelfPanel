WordPress Self Panel

Modern WordPress Management Tool for Termux

What is WordPress Self Panel?

WordPress Self Panel is a powerful, user-friendly tool designed to simplify WordPress management on Termux. It provides a comprehensive set of features for efficient website administration, all accessible through an intuitive interface.

Key Features

  • Easy installation and setup on Termux
  • Intuitive dashboard for website management
  • Built-in file manager with Tiny File Manager integration
  • Database management tools
  • SSL certificate management
  • Performance monitoring and optimization
  • Security tools and settings
  • Automatic updates for WordPress core, plugins, and themes
  • Email account management
  • Plugin and theme installation directly from the dashboard
  • Comprehensive logging system
  • Task management with undo/redo functionality
  • CLI interface for advanced users

Installation

To install WordPress Self Panel on Termux, run the following command:

wget -O install.sh https://raw.githubusercontent.com/likhonsheikhcodes/WordPressSelfPanel/main/install.sh && bash install.sh

Get Started

After installation, access your WordPress Self Panel by opening a web browser and navigating to http://localhost:8080. Log in with your credentials to start managing your WordPress site with ease!
